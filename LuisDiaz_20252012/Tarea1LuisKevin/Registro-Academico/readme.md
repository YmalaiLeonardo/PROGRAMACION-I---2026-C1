

📝 Descripción del Proyecto: Sistema de Registro y Evaluación Académica
Este proyecto es una aplicación de escritorio desarrollada en Java utilizando la biblioteca Swing para la interfaz gráfica de usuario (GUI). Su objetivo principal es agilizar e interactuar con el proceso de registro de estudiantes y el cálculo automático de sus calificaciones finales.

Características y Funcionalidades Clave:

Captura de Datos Personales: Permite registrar la información básica del estudiante, incluyendo Nombre Completo, Matrícula y Carrera a través de un formulario estructurado.

Procesamiento de Calificaciones: Cuenta con campos específicos para ingresar las notas de Prácticas, Parciales, Asignaciones y Examen Final.

Motor de Evaluación: El sistema suma las notas ingresadas y determina automáticamente el estado académico del alumno, indicando claramente si está Aprobado (60 puntos o más) o Reprobado (menos de 60 puntos).

Validación y Manejo de Errores: Incorpora bloqueos de seguridad para evitar que el programa falle si el usuario deja campos vacíos o intenta introducir letras en los campos de valores numéricos.

Arquitectura Orientada a Objetos: El proyecto aplica principios fundamentales de POO, separando la lógica de los datos (encapsulamiento mediante una clase modelo) del diseño visual de las ventanas, logrando un código limpio, modular y fácil de escalar.
