# Calculadora Matemática en Java

Programa de consola que hace sumas, restas, multiplicaciones y divisiones.

## Cómo usar:

### 1. Descargar el programa
- Descarga el archivo `CalculadoraMatematica.java`

### 2. Abrir la terminal (CMD en Windows, Terminal en Mac/Linux)

### 3. Compilar
Escribe esto en la terminal:
```bash`
javac CalculadoraMatematica.java


### 4. Ejecutar
Escribe esto en la terminal:
java CalculadoraMatematica

Ejemplo de uso:
-----CALCULADORA MATEMÁTICA-----

Ingresa el numero 1>> 10
Ingresa el numero 2>> 5

Menu de operaciones
1. Suma
2. Resta
3. Multiplicacion
4. Division
0. Salir
Selecciona una opcion>> 1

Resultado de la suma: 15.0
